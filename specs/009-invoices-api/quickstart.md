# Quickstart - Invoices API

## List invoices with optional company filter
```typescript
const invoices = await client.invoices.list({
  companyId: 'company-id',
  page: 0,
  size: 20
});

console.log(invoices.content[0]?.id);
console.log(invoices.page.totalElements);
```

## Get an invoice
```typescript
const invoice = await client.invoices.get('invoice-id');
console.log(invoice.total);
console.log(invoice.status);
console.log(invoice.dueDate);
```

## List invoice line items
```typescript
const items = await client.invoices.listItems('invoice-id', {
  page: 0,
  size: 100
});

console.log(items.content[0]?.productName);
console.log(items.content[0]?.quantity);
console.log(items.content[0]?.total);
```

## List draft invoice items for a company
```typescript
const drafts = await client.invoices.listDraftItems('company-id', {
  page: 0,
  size: 50
});

console.log(drafts.content[0]?.productName);
console.log(drafts.content[0]?.chargeType);
console.log(drafts.page.totalPages);
```

### Notes
- Pagination is page-based (`page`, `size` up to 200); `companyId` is the only supported filter for listing invoices.
- Invoice and draft items share the same shape; `invoiceId` is `null` for draft items.
- All operations are read-only; invoices are generated by Pax8 billing and cannot be created or updated via the client.
